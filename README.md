**О чем проект**
Cервисо Mesto: интерактивная страница, куда можно добавлять фотографии, удалять их и ставить лайки.

**Технологии**
1.Figma;
2. Grid Layout;
3. Media queries;
4.Git.
5.JS
Файловая структура организована по БЭМу.
Вёрстка адаптивная;
